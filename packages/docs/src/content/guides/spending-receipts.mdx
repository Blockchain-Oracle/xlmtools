# Spending Receipts

Every paid PULSAR tool call creates a real transaction on the Stellar testnet. You can verify any payment independently using public block explorers.

## Viewing your transaction history

### Option 1: Stellar Expert

1. Go to [stellar.expert/explorer/testnet](https://stellar.expert/explorer/testnet)
2. Paste your Stellar public key in the search bar
3. Navigate to the **Payments** or **Operations** tab
4. You'll see each USDC payment with amount, destination, and timestamp

### Option 2: Horizon API

Query the Stellar Horizon API directly:

```bash
curl "https://horizon-testnet.stellar.org/accounts/YOUR_PUBLIC_KEY/payments?order=desc&limit=10"
```

Replace `YOUR_PUBLIC_KEY` with your wallet address.

### Option 3: Ask your agent

```
Show me my PULSAR wallet and recent payments
```

The `wallet` tool returns your address and balance. You can then use the `stellar-account` tool to see full account details.

## Understanding the receipt

Each PULSAR payment transaction contains:

| Field | Description |
| --- | --- |
| **from** | Your wallet address |
| **to** | PULSAR API server's payment address |
| **amount** | Tool price in USDC (e.g. 0.003) |
| **asset** | USDC on Stellar testnet |
| **memo** | Tool name and request ID (e.g. `search:abc123`) |
| **timestamp** | When the payment was submitted |
| **tx_hash** | Unique transaction hash for verification |
