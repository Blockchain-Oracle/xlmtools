# FAQ

## General

### What is PULSAR?

PULSAR is a Stellar-native MCP (Model Context Protocol) server that gives AI agents access to 20 real-world tools. Seven tools are paid via USDC micropayments on Stellar, and fourteen are free.

### What is MCP?

MCP (Model Context Protocol) is an open standard that lets AI models call external tools. PULSAR implements the MCP server spec, so any MCP-compatible host (Claude, Cursor, Windsurf, etc.) can use PULSAR tools.

### What is MPP?

MPP (Micropayment Protocol) is a challenge-response payment protocol. When a paid tool is called, the server returns HTTP 402 with payment instructions. The client signs a Stellar USDC transfer and retries the request with the payment proof.

### Does PULSAR use real money?

No. PULSAR currently runs on the **Stellar testnet** only. All payments use testnet USDC, which has no monetary value. The transaction structure is identical to mainnet for demo purposes.

## Installation

### What are the system requirements?

- Node.js 22 or later
- An MCP-compatible host (Claude Code, Claude Desktop, Cursor, Windsurf, VS Code Copilot, Gemini CLI, OpenAI Codex, Zed, Continue, Cline, Goose) — or just use the standalone `pulsar-cli`
- Internet connection

### Do I need to install anything globally?

No. PULSAR runs via `npx`, which downloads and runs the package on demand. No global install needed.

### Where is my wallet stored?

Your Stellar keypair is stored at `~/.pulsar/config.json`. This file is created automatically on first run.

<Callout type="warning">
Your wallet file contains your Stellar secret key. While this is a testnet key with no real value, treat it as you would any private key.
</Callout>

## Pricing

### How much do tools cost?

| Tool | Price |
| --- | --- |
| search | $0.003 |
| research | $0.010 |
| youtube | $0.002 |
| screenshot | $0.010 |
| scrape | $0.002 |
| image | $0.040 |
| stocks | $0.001 |

All other tools (crypto, weather, domain, wallet, tools, and all 8 Stellar DEX tools) are **free**.

### Why are some tools free?

Free tools use public APIs with no upstream cost:

- **crypto** — CoinGecko public API
- **weather** — Open weather APIs
- **domain** — WHOIS lookup
- **Stellar tools** — Horizon, StellarExpert, and Reflector are all public Stellar infrastructure

### Is there a monthly fee or subscription?

No. PULSAR is strictly pay-per-call. No subscriptions, no minimums, no commitments.

## Payment

### How do I fund my wallet?

1. Get testnet XLM from [lab.stellar.org/account/fund](https://lab.stellar.org/account/fund)
2. Get testnet USDC from [faucet.circle.com](https://faucet.circle.com) (select Stellar Testnet)

See the [Fund Your Wallet](/getting-started/fund-wallet) guide for detailed steps.

### Is my secret key safe?

Yes. Your Stellar secret key is stored locally at `~/.pulsar/config.json` and **never leaves your machine**. Only signed transaction proofs are sent to the PULSAR API server.

### Can I verify payments on-chain?

Yes. Every paid tool call creates a real transaction on the Stellar testnet ledger. You can verify it on [Stellar Expert](https://stellar.expert/explorer/testnet) or via the Horizon API. See [Spending Receipts](/guides/spending-receipts) for details.

## Troubleshooting

### "npx: command not found"

Install Node.js 22+ from [nodejs.org](https://nodejs.org).

### My wallet balance shows 0

Your account may not be funded yet. Follow the [Fund Your Wallet](/getting-started/fund-wallet) guide.

### Tool calls hang or timeout

Check your internet connection. The PULSAR API server needs to be reachable. If running locally, make sure the API server is started.

### How do I reset my wallet?

Delete the wallet file and restart:

```bash
rm ~/.pulsar/config.json
```

A new keypair will be generated on the next tool call.
