# Tool Schemas

Each PULSAR tool is registered as an MCP tool with a typed input schema (using Zod) and structured output. Below are the complete schemas for all 21 tools.

<Callout type="info">
These schemas match the MCP `tools/list` response. Your MCP host uses these schemas to validate inputs before sending tool calls to PULSAR.
</Callout>

## Paid tools

### search

```typescript
{
  name: "search",
  title: "Web Search",
  inputSchema: {
    query: z.string(),          // Search query
    count: z.number().int()     // Results count (1-20, default 10)
      .min(1).max(20).default(10)
  }
}
```

### research

```typescript
{
  name: "research",
  title: "Deep Research",
  inputSchema: {
    query: z.string(),              // Research query
    num_results: z.number().int()   // Sources to analyze (1-20, default 5)
      .min(1).max(20).default(5)
  }
}
```

### youtube

```typescript
{
  name: "youtube",
  title: "YouTube Search / Lookup",
  inputSchema: {
    query: z.string().optional(),  // Search query
    id: z.string().optional()      // Video ID for direct lookup
  }
}
```

### screenshot

