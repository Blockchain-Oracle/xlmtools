# PULSAR

**Stellar-native MCP tools with pay-per-call micropayments.**

PULSAR is a Model Context Protocol (MCP) server that gives AI agents access to 20 real-world tools — web search, image generation, financial data, Stellar DEX analytics, and more. Every paid tool call is settled instantly on the Stellar network using USDC micropayments via the Micropayment Protocol (MPP).

One command to install. No API keys. No subscriptions. Just pay-per-call.

```bash
claude mcp add pulsar npx @pulsar/mcp
```

---

## Why PULSAR?

- **No API keys to manage** — PULSAR handles all backend integrations. You just call tools.
- **Pay only for what you use** — Micropayments as low as $0.001 per call. No monthly fees.
- **Stellar-native** — Payments settle in seconds on the Stellar network using USDC.
- **21 tools in one server** — Web, media, finance, and the full Stellar DEX — all from a single MCP install.
- **Works with any MCP host** — Claude Code, Claude Desktop, Cursor, Windsurf, and more.
