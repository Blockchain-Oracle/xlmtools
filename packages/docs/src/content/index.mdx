# XLMTools

**Stellar-native MCP tools with pay-per-call micropayments.**

XLMTools is a Model Context Protocol (MCP) server **and** a standalone CLI **and** a cross-client Agent Skill. It gives AI agents access to 21 real-world tools — web search, image generation, financial data, Stellar DEX analytics, and more. Every paid tool call is settled instantly on the Stellar network using USDC micropayments via the Micropayment Protocol (MPP).

Works natively in Claude Code, Claude Desktop, Cursor, Windsurf, VS Code Copilot, Gemini CLI, OpenAI Codex, Zed, Continue, Cline, and Goose.

```bash
# Standalone CLI (universal — works with any agent host)
npm install -g @xlmtools/cli

# MCP server (optional fast-path for Claude Code, Cursor, Cline, etc.)
claude mcp add xlmtools npx @xlmtools/mcp

# Agent Skill (install via prompt in any agent)
Read https://xlmtools.com/skill.md and follow the instructions to install XLMTools.
```

## Why XLMTools?

- **No API keys to manage** — XLMTools handles all backend integrations. You just call tools.
- **Pay only for what you use** — Micropayments as low as $0.001 per call. No monthly fees.
- **Stellar-native** — Payments settle in seconds on the Stellar network using USDC.
- **21 tools in one server** — Web, media, finance, and the full Stellar DEX — all from a single install.
- **Works with every major client** — Claude Code, Cursor, Windsurf, VS Code Copilot, Gemini CLI, OpenAI Codex, Zed, Continue, Cline, Goose, and more.
- **Three install flavors** — MCP server, standalone CLI, or cross-client Agent Skill.
