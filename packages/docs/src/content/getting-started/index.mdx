# Getting Started

Get PULSAR running in your AI agent in under 2 minutes. This guide covers installation, wallet setup, and making your first paid tool call.

## Steps

1. [Installation](/getting-started/installation) — Add PULSAR to your MCP host.
2. [Fund Your Wallet](/getting-started/fund-wallet) — Get testnet USDC to pay for tool calls.
3. [First Tool Call](/getting-started/first-tool-call) — Make a search and see the payment flow in action.

## Prerequisites

- **Node.js 22+** — PULSAR runs as a local stdio MCP server via `npx`.
- **An MCP-compatible host** — works with 12+ clients: Claude Code, Claude Desktop, Cursor, Windsurf, VS Code Copilot, Gemini CLI, OpenAI Codex, Zed, Continue, Cline, Goose. Or skip the host entirely and use the standalone `pulsar-cli`.
- **A Stellar testnet wallet** — PULSAR auto-generates one on first run and auto-funds it with XLM via friendbot. You only need to grab testnet USDC from faucet.circle.com.
