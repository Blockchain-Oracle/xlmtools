# Weather

Get current weather conditions and temperature for any city worldwide.

**Price:** Free

## Parameters

| Parameter | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `location` | string | Yes | — | City name (e.g. `Lagos`, `London`, `New York`) |

## Example

```
What's the weather in Lagos?
```

## Response

Returns current conditions:

- **location** — City and country
- **temperature** — Current temperature
- **conditions** — Weather description (e.g. "Partly cloudy")
- **humidity** — Relative humidity
- **wind** — Wind speed and direction

## Use cases

- Quick weather checks during travel planning
- Adding real-time weather context to conversations
- Comparing conditions across cities
