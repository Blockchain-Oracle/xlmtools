# Image Generation

Generate images from text prompts using AI. Returns high-resolution images in multiple aspect ratios.

**Price:** $0.040 USDC per call

## Parameters

| Parameter | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `prompt` | string | Yes | — | Text prompt describing the image to generate |
| `size` | enum | No | `1024x1024` | Output dimensions: `1024x1024`, `1024x1792`, or `1792x1024` |

## Example

```
Generate an image of a futuristic cityscape with Stellar blockchain nodes orbiting overhead, 1792x1024
```

## Size options

| Size | Aspect ratio | Best for |
| --- | --- | --- |
| `1024x1024` | 1:1 | Profile images, icons, social posts |
| `1024x1792` | 9:16 | Phone wallpapers, stories, vertical banners |
| `1792x1024` | 16:9 | Desktop wallpapers, hero images, presentations |

## Response

Returns the generated image data that your MCP host can display inline.

## Use cases

- Creating illustrations for documents and presentations
- Generating concept art and mockups
- Producing social media visuals on the fly
- Rapid prototyping of design ideas
