## How payment works

When you call a paid tool, PULSAR handles the payment automatically:

1. The API server returns HTTP `402 Payment Required`
2. The MPP client builds a Soroban USDC transfer transaction
3. Your local Stellar key signs it
4. The request is retried with payment proof
5. The API verifies the payment and returns the result

No manual steps required. See the [Payment Flow](/guides/payment-flow) guide for details.
